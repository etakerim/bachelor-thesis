◆ process_correlation()

void process_correlation ( uint8_t  axis,
const float *  buffer,
Statistics *  stats,
Correlation *  corr,
const SamplingConfig *  c 
)

Korelácia medzi osami akcelerácie: XY, XZ, YZ. Dochádza k bariérovej synchronizácii!

Úloha pre každú os zrýchlenia si nezávisle prepočíta rozdiely vzoriek od priemeru a smerodajné odchýlky. Následne dochádza k bariérovej synchronizácii aktívnych osí. Každá úloha si dopočíta všetky korelácie samostatne.

Parametre
[in]axisOs akcelerácie: 0, 1, 2
[in]bufferPosuvné okno vzoriek signálu
[out]statsZistené medziosové korelácie
[out]corrPomocné polia pre výmenu predspracovaných údajov medz úlohami (osami)
[in]cNastavenia vzorkovania. Využíva sa dĺžka posuvného okna a povolené osi.
